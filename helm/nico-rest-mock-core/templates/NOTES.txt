1. Get the mock service endpoint:

  export POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "nico-rest-mock-core.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -o jsonpath="{.items[0].metadata.name}")
  kubectl --namespace {{ .Release.Namespace }} port-forward $POD_NAME 11079:{{ .Values.service.port }}

2. The Forge gRPC server listens on port {{ .Values.service.port }}.

3. Edit mock hosts in Helm values key `inventory` (`helm/nico-rest-mock-core/values.yaml`).
4. When `libvirt.enabled` is true, only machines whose `id` matches an existing libvirt domain are returned.
5. Set `libvirt.volumeGiB` to change the default root disk size for provisioned instances (default: 20).
